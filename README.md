# Basaveshwar Hotel Management System

A premium, modern web application for Basaveshwar Hotel, featuring a 2026-standard UI/UX, online pre-ordering, and multi-language support.

## Features
- **Premium UI**: Glassmorphism, smooth animations, and interactive elements.
- **Multi-language**: Support for English, Hindi, and Kannada.
- **Online Pre-order**: Seamless cart system and pickup scheduling.
- **Admin Dashboard**: For managing orders, menu, and occupancy.

## Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JS.
- **Backend**: Node.js, Express.
- **Database**: MySQL.

## Local Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up MySQL database `basaveshwar_hotel` and import `database/schema.sql`.
4. Configure `.env` file with your database credentials.
5. Run the server: `npm start`.
